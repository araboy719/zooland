Структура папок и файлов
=======================================
index.php - главная страница приложения
/ assets - настройки с Bootstrap  
	fonts
	js
/configs - все конфигурации для приложения
	db.php - подключение к базе данных
/css - стили для блоков сайта
	style.css - 
	style_zoo.css - наши стили
/images - набор фото и картинок
    slider - наши картинки
/parts - части сайта
	footer.php - нижняя часть сайта
	header.php - верхняя часть сайта
	animals.php - наши животные
	contacts.php - наши контакты, карта
	helpers.php - наши опекуны
	servises.php - наши услуги
exit.php - выход
login.php - авторизация
register.php - регистрация
